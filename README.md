# AutoClicker-Go

**AutoClicker-Go** is a simple, lightweight auto-clicker desktop application written in **Go**, built with **Fyne** for the UI and **RobotGo** for mouse automation.  
It was originally created for fun and to help a friend play Minecraft Bedwars more easily by automating mouse clicks.
